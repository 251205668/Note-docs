# 深浅拷贝
## 浅拷贝

只是复制栈的内存地址，如果另一个对象改变堆的值，本身会发生改变

1. 引用复制

```js
function shallowCopy(target){
  const res = {}
  for(let key in target){
    res[key] = target[key]
  }
  return res
}
```

2. Object.assign

```js
var x = {
  a: 1,
  b: { f: { g: 1 } },
  c: [ 1, 2, 3 ]
};
var y = Object.assign({}, x);
console.log(y.b.f === x.b.f);     // true

```
